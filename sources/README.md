# Sources

The active tree stores only source material needed for current inquiries and
only to the extent lawfully and practically reproducible.

Each inquiry's `evidence.yaml` identifies exact paths, SHA-256 values, roles, and
limitations. Excerpt bundles do not silently become complete witnesses.

Full source files that cannot lawfully or practically be stored here must have a
recoverable bibliographic and archival locator before claims depending on them
are promoted.
