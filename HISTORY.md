# History

The repository was streamlined under the owner's explicit authorization after
its active tree accumulated competing instructions, amendments, manifests,
registers, certifications, compatibility gates, and duplicate lifecycle records.

The complete pre-rewrite repository is preserved at:

- commit `774ffea9d05d626acefae75a9b97841681bdd3c8`;
- branch `archive/pre-streamline-custos-2026-07-23`.

Those records remain available for historical and forensic study. They do not
govern the current working tree unless a current file explicitly imports them.

The rewrite preserves the substantive method while replacing simulated
institutional bureaucracy with one active instruction, one protocol, one current
record per inquiry, automatic Git provenance, and ordinary software tests.
