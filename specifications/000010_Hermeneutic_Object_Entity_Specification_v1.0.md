PURPOSE

This Specification defines native admission requirements for Hermeneutic Object entities while preserving existing constitutional distinctions and jurisdictional boundaries.

Admission of this Specification does not admit any Hermeneutic Object, create the Hermeneutic Object Register, or amend Documents 000001 through 000005.

I. CONSTITUTIONAL BASIS AND SCOPE

1. This Specification is subordinate to Documentary Ontology, Identifier Specification, Relationship Specification, Controlled Vocabulary Specification, and the Constitutional Admission Protocol read with AMD-000000002.
2. Where this Specification conflicts with an admitted constitutional specification, the admitted constitutional specification prevails.
3. CAG-000000006 admitted this Specification through DDR-000000002 after distinct determinations by CAG-000000001 through CAG-000000004.
4. This Specification governs future native-class Hermeneutic Object admission packages only after the required Hermeneutic Object Register has been separately constructed and admitted.

II. NATIVE CLASS DISTINCTIONS (PRESERVED)

The following distinctions remain mandatory:

1. HO = general Hermeneutic Object.
2. HOA = Hermeneutic Object A (primary-source-centered).
3. HOB = Hermeneutic Object B (second-author/comparison-centered).
4. HOC = Hermeneutic Object C (Strauss-centered).

A class designation shall not be collapsed for convenience, and subtype distinctions shall not be replaced by generic labeling when subtype determination is documentarily supportable.

III. DEFINITION

A Hermeneutic Object is a bounded interpretive documentary artifact preserving an explicit inquiry package, its documentary basis, and its uncertainty state without conflating documentary evidence with interpretive conclusion.

IV. NATIVE ADMISSION REQUIREMENTS

A candidate HO/HOA/HOB/HOC may be admitted in its native class only when all required conditions are satisfied:

1. Class conformity
   Proposed class is one of: Hermeneutic Object, Hermeneutic Object A, Hermeneutic Object B, Hermeneutic Object C.
2. Documentary boundedness
   Inquiry scope is explicitly bounded and recoverable.
3. Source inquiry traceability
   Source inquiry and governing question are identified.
4. Documentary basis stated
   Documentary basis is explicit, recoverable, and auditable.
5. Claims and questions explicit
   Claims and unresolved questions are separately stated.
6. Provenance completeness
   Provenance of reconstruction and dependencies is preserved.
7. Status truthfulness
   Reconstruction status and uncertainty are explicitly and non-misleadingly recorded.
8. Governance traceability
   Responsible authority and governing references are recorded.
9. Historical integrity
   Version history and audit history are present and append-preserving.
10. Admission traceability
   Admission record reference is present.
11. Certification boundary clarity
   Certification eligibility and engine-use authorization state are explicit.

V. MINIMUM CANONICAL OBJECT FIELDS

Every admitted native-class Hermeneutic Object record shall contain at minimum:

1. identifier
2. class
3. title
4. bounded inquiry scope
5. source inquiry
6. documentary basis
7. claims
8. questions
9. provenance
10. reconstruction status
11. uncertainty
12. lifecycle status
13. responsible authority
14. governing references
15. version history
16. audit history
17. admission record reference
18. certification eligibility
19. engine-use authorization state

A field may be internally structured, but each listed element is required as a canonical datum.

VI. OPEN DOCUMENTARY RECONSTRUCTION ADMISSION

Native admission of an open documentary reconstruction is permitted when all of the following are true:

1. The object record itself is constitutionally complete as a native-class record.
2. Unresolved inquiry is clearly stated in questions and uncertainty fields.
3. No claim of completed reconstruction is made.
4. Certification remains withheld.
5. Engine-use authorization remains withheld.

Open reconstruction admission preserves a valid documentary state without falsely declaring closure.
